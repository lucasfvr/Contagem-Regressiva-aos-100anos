# Contagem Regressiva aos 100 Anos#

# Descrição do programa:

- Esse programa foi desenvolvido para descobrir quantos anos faltam para você completar 100 anos.

# Requisitos para executar o programa:

- Você deve ter o Python instalado em seu computador para executar este programa. A versão mínima recomendada é o Python 3.x.

# Como executar o programa:

- Certifique-se de que o Python está instalado em seu sistema. Caso não esteja, você pode baixá-lo em python.org;
- Abra o Prompt de Comando;
- Pressione as teclas 'Windows' + 'R' simultaneamente;
- Na caixa de diálogo "Executar", digite 'cmd' e pressione 'Enter';
- Localize o diretório onde o programa foi salvo. Exemplo: Se o programa estiver salvo na área de trabalho, digite o comando: cd Desktop;
- No Prompt de Comando, digite o seguinte comando para executar o programa:python "contagem regressiva 100 anos.py"
- Siga as orientações exibidas pelo programa, respondendo às perguntas sobre seu nome e idade, o programa calculará automaticamente quantos anos faltam para você completar 100 anos.
